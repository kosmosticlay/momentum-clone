    📋타이머 사용방법
    * 초기 세팅 *

    ① 원하는 시간을 입력 후 SET 버튼을 누른다.
    (세팅시간 텍스트로 시간 확인 가능)
    ② START 버튼을 누르면 타이머가 작동을 시작한다.
    ③ STOP 버튼으로 타이머를 일시정지할 수 있다.
    (STAET버튼을 누르면 일시정지한 지점에서 다시 작동한다.)
    ④ RESTART 버튼으로 세팅된 시간의 타이머를 다시 시작할 수 있다.

    * 세팅시간 변경 *

    ① RESET 버튼으로 세팅시간을 초기화 한다.
    (RESET버튼을 누르지 않고 시간을 변경하면 정상적으로 작동하지 않는다.)
    ② '초기 세팅'과 같은 방식으로 타이머 시간을 설정한다.
